UI Mihomo / Ultra — обновление переноса v1.2.113

Что изменилось в этом релизе:
- `v1.2.113` продолжает оптимизацию именно роутерной версии, без возврата тяжёлого Mihomo-конфига в активный UI
- router-agent разделён на два уровня статуса: лёгкий `status` для частого polling и отдельный `status_debug` для storage / firmware / model / kernel / xkeen / Mihomo / shaper diagnostics
- `SystemCard` теперь обновляет live-ресурсы отдельно от тяжёлой отладочной информации, поэтому Overview больше не платит полную цену за каждое обновление
- фронтенд получил отдельный `agentStatusDebugAPI()` с длиннее живущим cache, чтобы тяжёлые поля не дёргались так же часто, как CPU/RAM/uptime
- router-agent обновлён до `0.6.26`

Текущие версии:
- UI: v1.2.113
- router-agent: 0.6.26

Что важно дальше:
- базовый путь роутерного UI должен оставаться дешёвым: status/traffic/qos без тяжёлой диагностики по умолчанию
- следующие релизы лучше продолжать как lazy/debug-only оптимизацию: редкие сетевые/шейперные данные по кнопке, а не в постоянном фоне
