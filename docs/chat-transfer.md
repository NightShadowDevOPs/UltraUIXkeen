UI Mihomo / Ultra — обновление переноса v1.2.90

Что изменилось в этом релизе:
- в `Mihomo → Конфигурация` добавлен отдельный редактор `rule-providers` поверх текущего YAML в редакторе
- `rule-providers` теперь можно редактировать по одному, дублировать, добавлять из шаблона и отключать с безопасной подчисткой связанных `RULE-SET` правил в секции `rules`

- `router-agent` в этом релизе не менялся; линия остаётся `0.6.22`
