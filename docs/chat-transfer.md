UI Mihomo / Ultra — обновление переноса v1.2.82

Что изменилось в этом релизе:
- в `Mihomo → Конфигурация` добавлен структурированный обзор популярных секций `config.yaml`
- обзор работает для `active / draft / baseline / editor` и показывает режим, порты, controller, состояние модулей `tun / dns / profile / sniffer`, список top-level секций и объём `proxies / groups / rules / providers`
- `router-agent` в этом релизе не менялся: остаётся `0.6.21`
- line-diff из v1.2.80, диагностика операций из v1.2.81 и автоперепроверка статуса `Router → Agent` из v1.2.77 сохранены

Текущие версии:
- UI: v1.2.82
- router-agent: 0.6.21

Что важно по интерфейсу:
- `Mihomo` остаётся отдельным рабочим разделом
- `Settings` держит только интерфейс, поведение и связанные настройки отображения
- внутренняя навигация `Mihomo` осталась прежней по логике, менялись только русские подписи и формулировки

Следующий логичный шаг:
- после v1.2.82 можно идти либо в просмотр последней успешно применённой версии, либо в более умный редактор частых секций `config.yaml`
