31.03.2026 UI Mihomo / Ultra — обновление переноса v1.2.111

Что изменилось в этом релизе:
- начат целевой этап оптимизации роутерной версии UI без возврата тяжёлого Mihomo workspace
- скрытые секции страницы `Router` больше не живут через `v-show`: теперь они размонтируются и не продолжают фоновые polling-циклы
- polling для `SystemCard`, `AgentCard`, host QoS, user QoS, `NetcrazeTrafficCard` и блока Tasks/logs стал реже и дополнительно гасится на скрытой вкладке браузера
- `NetcrazeTrafficCard` ослабил частоту опроса router/host traffic, QoS и LAN-host refresh, чтобы снизить CPU churn на роутере
- `router-agent` в релизе не менялся и остаётся `0.6.24`

Текущие версии:
- UI: v1.2.111
- router-agent: 0.6.24

Что важно по текущему состоянию:
- это первый релиз именно по оптимизации router UI, а не по развитию новых функций
- раздел `Mihomo` уже убран из роутерной версии и не должен возвращаться до завершения общей оптимизации
- текущий приоритет: снизить фоновую нагрузку UI/agent, оставить только полезный runtime-control, и уже после этого точечно дорабатывать shaping/QoS
